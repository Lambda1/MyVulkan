My Vulkan
--

1. リポジトリ説明
	Vulkanを学習するために作成.
	主に, 自作ラッパやテストプログラム, ドキュメントの管理を行う.

2. ディレクトリ
	document: Vulkan Tutorial[1]のメモを日本語で作成したものや個人的なメモなどのドキュメント.
	template_sln: Visual Studio 2019でVulkanを扱うための最小テンプレートファイル.

0. 開発環境
	OS:  Windows10 Home
	IDE: Visual Studio 2019
	SDK: VulkanSDK 1.2.131.2
	Lib: GLFW, GLM, C++標準ライブラリ

参考文献
	本プログラムを作成するにあたり, 以下の記事を参考にさせていただきました.
	[1] Vulkan Tutorial (https://vulkan-tutorial.com)
